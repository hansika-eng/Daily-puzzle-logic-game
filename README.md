# Daily Puzzle Logic Game

A daily puzzle-based logic game built as a capstone project.

## Tech Stack
- React + Vite + TypeScript
- Tailwind CSS
- Firebase Authentication (planned)
- PostgreSQL (Neon) (planned)
- IndexedDB for client storage
